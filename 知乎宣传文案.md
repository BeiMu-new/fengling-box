# 枫铃工具箱 fengling-box — 开源多功能命令行工具箱

做过 AI Agent 的人应该都有过这个痛点：想搜个资料，得在十几个来源之间来回切换，每个平台 API 格式还不一样，配置起来特别烦。

所以我做了 **fengling-box（枫铃工具箱）**，一个命令行多功能工具箱，命令统一用 `flb` 调用，开箱即用。

## 核心功能

### 1. 多源聚合搜索（42个搜索源）

用户显式选择搜索源，不自动路由，你想用哪个就用哪个：

- **通用**：百度、Bing、DuckDuckGo、Brave、Yandex、Google、Searx
- **AI聚合**：AnySearch、Tavily、Exa
- **中文平台**：微博、知乎、B站、抖音、今日头条
- **新闻媒体**：新华网、央视网、澎湃新闻、36氪、虎嗅、IT之家、财联社
- **技术**：GitHub、npm、PyPI、Stack Overflow、crates.io、pkg.go.dev
- **论坛**：Reddit、Hacker News
- **学术**：arXiv、维基百科、Semantic Scholar、Google Scholar
- **计算**：Wolfram Alpha
- **电商**：淘宝、京东、拼多多
- **娱乐**：豆瓣、IMDb
- **百科**：百度百科、萌娘百科

不需要 Key 的免费直接用，需要 Key 的有详细文档告诉你怎么申请（大部分都有免费额度）。

### 2. 文件批处理

批量重命名、批量复制、文件列目录，支持前缀/后缀/序号补零/扩展名过滤/大小写转换/字符串替换，还有预览模式。

### 3. 图片处理

压缩、格式转换（jpg/png/webp/avif/tiff）、调整尺寸、文字水印、图片信息查看。基于 sharp，质量有保证。

### 4. 数据处理

CSV 选列/过滤/导出、Excel 读取转 CSV、文本文件清洗（去重/去空行/大小写/替换）、JSON 格式化/查询/提取。

### 5. 信源核实

输入 URL，自动匹配可信/不可信/争议信源名单，标注来源类型和可信度等级。做新闻核实、事实核查的时候特别好用。

### 6. 免费翻译

Google 翻译接口，失败自动降级到 MyMemory，支持几十种语言。

### 7. 其他

搜索历史记录、多引擎并行搜索、结果导出（JSON/Markdown/TXT）、URL 一键打开。

## 安装

```bash
npm install -g fengling-box
```

## 更新

```bash
npm install -g fengling-box@latest
```

## 使用示例

```bash
# 搜索
flb search baidu "AI大模型"
flb search github "vue" --limit 5
flb search arxiv "speculative decoding" --out result.md
flb search all "OpenCode" -e baidu,bing,github,reddit

# 文件
flb file rename ./photos --prefix "trip_" --start 1
flb file list ./folder --ext jpg,png --size

# 图片
flb image compress ./photos --quality 80
flb image convert ./photo.png --to webp

# 数据
flb data csv ./data.csv --info
flb data excel ./data.xlsx --out result.csv

# 信源核实
flb verify https://www.xinhuanet.com/xxx

# 翻译
flb translate "hello world" --to zh
```
